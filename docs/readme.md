# zipfai
Zipfai is an attempt to prove Zipf's Law and implement it to generate text using AI.  

## Requirements  
- Python 3
- Pip

## Installation
Windows: 
```
git clone https://github.com/cbracketdash/zipfai.git
cd zipfai
build.bat
```
Mac/Linux:
```
git clone https://github.com/cbracketdash/zipfai.git
cd zipfai
sh build.sh
```
